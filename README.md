# Angular8 - CURD 

# Get the code

$ git clone https://github.com/Subash-Rathnayaka/Angular8-CURD.git
$ cd angular8-crud
$ npm install

# Run the app

$ ng serve

# Development

Run ng serve for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.
